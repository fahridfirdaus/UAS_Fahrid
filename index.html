Sebuah web server adalah perangkat keras atau perangkat lunak yang menyimpan, memproses, dan menyajikan halaman-halaman web kepada pengguna. Ketika seseorang mengakses sebuah website, permintaan dikirimkan ke web server yang menyimpan halaman-halaman tersebut. Web server menanggapi permintaan dengan mengirimkan data (berupa halaman web) ke perangkat pengguna melalui internet. Domain: Domain adalah alamat unik yang digunakan untuk mengidentifikasi lokasi suatu website di internet. Contohnya adalah www.nama-website.com. Domain memudahkan pengguna untuk mengingat alamat website tanpa harus menghafal alamat IP numerik yang sebenarnya digunakan oleh web server. Proses penerjemahan dari nama domain ke alamat IP disebut DNS (Domain Name System). DNS membantu mengarahkan permintaan pengguna ke server yang benar
